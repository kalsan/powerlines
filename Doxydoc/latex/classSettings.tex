This class provides a simple interface for dealing with settings.

First, a \hyperlink{classSettings}{Settings} object must be created, specifying a file name. If you create multiple such objects, you\textquotesingle{}re dealing with completely independent settings sets, using different files. Second, all settings to be taken care of must be made known by calling add\+Setting. You must provide\+: -\/ an identifier string, e.\+g. \char`\"{}gen\+Font\+Size\char`\"{},
\begin{DoxyItemize}
\item a description that will be written to the settings file, e.\+g. \char`\"{}\+Set Generel Font Size here.\char`\"{}
\item a default integer value used for initializing the setting, when resetting it, or when we fail loading it from the file. e.\+g. 12 To deal with the value of a setting, use \hyperlink{classSettings_a2ed21ce3e76ab2c34ac8684c26ae951f}{set()}, \hyperlink{classSettings_a8976b08faf595d3da853e59d4f79e45d}{reset()} and \hyperlink{classSettings_ae2aff237502f56739fff3d664cb0fa99}{get()}, specifying the setting\textquotesingle{}s identifier. To reset all values to their default, use \hyperlink{classSettings_a921ad66038065757097cb9d199bedde9}{reset\+All()}. To read / write all the settings handled by your \hyperlink{structSetting}{Setting} object from / to disk, use \hyperlink{classSettings_a2042818d2c6767f73c5c4e9eeac23b61}{load()} / \hyperlink{classSettings_ab017f1e06b354b016ade4f17637397be}{save()}. If you want to avoid that missing lines are automatically appended to the file, use \hyperlink{classSettings_a8bfa41f6c8f6eb48d116f96115d9793d}{load\+No\+Fix()}.
\end{DoxyItemize}
