Detailed Description

This class allows to simultaneously draw to screen and register to evilPixels. It takes care of the management of any "dangerous" object on the screen during the game.

bool* GameScreen::evilPixels
private

Datastructure used for collision detection. This has one entry for each pixel on the screen. If a player lands on a pixel where evilPixels is true, a collision is caused and the player dies. Order: height * prgw + width (left-to-right, top-to-bottom)
